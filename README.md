## Overview

In this React application, the advantages of GraphQL technology over REST API architecture are highlighted and demonstrated practically. Additionally, many functions used in GraphQL technology are discussed. These functions can be listed as follows:

- Cache Mechanism
- Real-Time Data Processing
-  GraphQL allows clients to request exactly the data they need, avoiding over-fetching or under-fetching .
- Unlike REST, which typically requires multiple endpoints, GraphQL uses a single endpoint for all data queries.
